About Administrator Account

The Administrator is Responsible for Creating an Authorized User Account to access the System and as well as Revoke an Account or Update the Information. With that being said, the Admin's task is not much but the one maintains an account for the user

User Management / User Creation
  • Log-in as Admin
  • Click User Management to view, update, and delete the desired account(As shown)
  • Click User Creation to Create a new User Account and give them access to the system as you see fit or limit the access of info allowed to the user(As shown)
  • To Create a new User Account, Click User Creation
